# 🏛️ Fountain Johannesburg — Luxury Hospitality & Wellness Platform

> A premium hotel and wellness website experience built with modern frontend technologies, focused on luxury presentation, performance, and smooth user interaction.

---

## Overview

Fountain Johannesburg is a fictional luxury hospitality platform designed to showcase how high-end brands can combine immersive visual design with fast, scalable web architecture.

The project simulates a 5-star hotel experience featuring:

- Luxury dining experiences
- Wellness and spa services
- Editorial-style storytelling
- Cinematic layouts and interactions
- Production-grade frontend engineering

Built primarily for frontend excellence, the platform demonstrates how modern web technologies can create emotionally engaging user experiences without sacrificing performance.

## Performance & Engineering

This project was engineered with production-quality frontend standards in mind.

### Fast & Stable Experience

- Optimized animations using `framer-motion`
- Smooth cubic-bezier motion system
- Consistent 60 FPS interactions

### Zero Layout Shift (CLS)

Images use responsive aspect-ratio containers with Next.js image optimization to prevent layout jumping during loading.

### Scalable Frontend Architecture

- Component-driven structure
- Data-driven UI rendering
- Reusable design systems
- Cleaner state and rendering performance

### Asset Reliability

External media dependencies were cleaned and standardized to reduce broken assets and improve long-term maintainability.
